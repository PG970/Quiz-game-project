# Quiz-game-project
## **1. Project Overview**

This project is a simple **Python-based Quiz Game** where the user is asked multiple-choice questions (MCQs).
The game displays each question with four options and allows the player to input their answer.
If the player selects the correct answer, they win a prize amount. If incorrect, the game stops.

This project demonstrates the use of:

* Lists
* Loops
* Conditional statements
* User input
* Basic game mechanics
